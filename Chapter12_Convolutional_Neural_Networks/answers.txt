Mixed Precison

4. Why would you want to add a max pooling layer rather than a convolutional
layer with the same stride?

max pooling needs no weights, reduces overfitting, Strong invariance

5. When would you want to add a local response normalization layer?
Use LRN to boost strong activations and suppress weaker neighboring ones, though it’s mostly outdated today.

6. Can you name the main innovations in AlexNet, compared to LeNet-5? What
about the main innovations in GoogLeNet, ResNet, SENet, and Xception?

AlexNet: ReLU, Dropouts, GPU training
GoogLeNet: 1x1 conv for dimension reduction
ResNet: Residuals
SENet: Channel attention
Xception: Depthwise separable convolutions

7. What is a fully convolutional network? How can you convert a dense layer into a
convolutional layer?

FCN: only convolutional layers(no denser layers) outputs a spatial map instead of a single vector.
User a Full size filter.

8. What is the main technical difficulty of semantic segmentation?
The main difficulty is making accurate pixel-level predictions while preserving spatial detail.
